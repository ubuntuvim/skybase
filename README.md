# skybase
一个提供数据服务的框架。

主要包括2大块内容：

1. 类似于Struct2的请求分发模板
2. 类似于Spring IOC容器，用于实例化类之间的关系

有关请求分发请看[https://github.com/ubuntuvim/myMVC](https://github.com/ubuntuvim/myMVC)。
